<h3>1. 프로젝트 소개</h3>
racconworld는 누구나 무료로 즐길 수 있는 Test 기반 웹 사이트입니다.
<br>
이 프로젝트의 목표는 단 하나 잠깐이나마 사랑하는 연인, 친구, 가족들과 결과를 공유하며, 함께 소소한 재미를 느끼는 순간을 만드는 것입니다.
